# espresso-walk

A terminal application that generates a physics-based 3D orbital color walk anchored around espresso brown

## Usage

```text
Usage: espresso-walk [options]
Runs a physics-based 3D orbital color walk anchored around espresso brown

Options:
  -gap
        Add a newline gap between color bars
  -steps int
        Number of steps in the walk (default 15)
```
